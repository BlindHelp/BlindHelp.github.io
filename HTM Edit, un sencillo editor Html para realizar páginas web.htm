Nota muy importante:
Una vez que hayas salvaguardado y que piensas que no hay más cambio para hacer en este archivo en html abrelo con el bloc de notas de Windows, busca el elemento: guarde como y luego seleccione la codificación: UTF-8
Lamentablemente, el programa HTM EDIT lo guarda con la codificación: ANSI
Si alojó el archivo en cualquier plataforma usando un enlace, tus amigos verán signos cabalísticos en lugar de acentos.
Tal vez también puede ocurrir cuando lo abres localmente desde tu computadora.
A veces el texto es legible porque lo abres localmente. Pero insisto en que debes volver a guardarlo como se indicó anteriormente.
