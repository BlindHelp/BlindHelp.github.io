A propos de la traduction des modules complémentaire et comment faire pour le transférer à la Communauté (pour développeurs et traducteurs).

Voici quelques informations en anglais a tout fin utile:
NVDA Translation documentation
Please find the latest documentation on the NVDA wiki:
http://nvda-project.org/wiki/TranslatingNVDA
Sur cette page en anglais on vous explique comment être un membre de la communauté de traducteurs pour NVDA y compris la traduction pour les modules complémentaires.

Ici, il y a une autre page en anglais qu'il peut vous être utile :
Translate – NVDA Community
http://community.nvda-project.org/wiki/Translate

celleci aussi:
http://community.nvda-project.org/wiki/TranslatingUsingAutomaticProcess

Vous devez avoir un logiciel appelé :
TortoiseSVN TortoiseSVN - Downloads C'est pour récupérer le dépôt SVN = (SVN repository ) de NVDA.
N'oubliez pas de consulter se lien afin de savoir comment recevoir une invitation et être membre de la communauté de traducteurs pour NVDA
http://nvda-project.org/wiki/TranslatingNVDA En revanche, vous devriez avoir un logiciel appelé Poedit. Poedit: Download Le quelle vous servira pour récupérer les messages depuis les fichiers .py du module complémentaire. Vous devez les déclaré avant dans le code pour être exportées avec poedit.
Voir les tutoriels réalisés par mes soins qui expliquent tout cela.
Je pense que la personne qui a fait le module complémentaire doit l'avoir fait, si se n'est pas le cas vous devriez contacter la personne qui a fait le module complémentaire et lui dire que vous êtes intéressée à traduire son module complémentaire.

Infos practiques:

Joseph Lee, qui est le traducteur de NVDA en coréen, puis créateur de la liste des modules complémentaires et également auteur de certains modules complémentaires, a envoyer cet e-mail à la liste de programmation (ou développement) de NVDA en anglais.
Il communique l'existence d'une liste pour apprendre Python, le langage fondamentale pour la programmation de NVDA:
Voici l'info en anglais:
A new list about learning Python for visually impaired people

Hi all,

Somebody sent a forwarded message to the program-l list about a new mailing list for learning Python. Since NVDA is written in Python, I thought I'd pass this info to you so you can lend some support there.

The subscription address is:

pythonvis-request@freelists.org

Subject is subscribe (no confirmation necessary). Thanks.

JL

Traduction:

Une nouvelle liste sur l'apprentissage de Python pour les personnes ayant une déficience visuelle
Salut a tous, Quelqu'un a envoyé un message transféré à la liste de programme-l sur une nouvel liste de diffusion pour l'apprentissage de Python. Étant donné que NVDA est écrit en Python, j'ai pensé que je pourrais passer cette info pour vous afin que vous pouviez apporter quelque soutien là.
L'adresse d'abonnement est:
pythonvis-request@freelists.org

Il faut envoyer un message vide comme objet:
subscribe
(aucune confirmation nécessaire). Merci.

JL
