FreeTube

On ne s’embarrasse plus avec Youtube car il existe FreeTube !
Depuis la mise en place de plus forte restriction au niveau des bloqueurs de pubs sur YouTube, on entend beaucoup parler de méthode de contournement parfois les plus insolites les unes que les autres (changement de bloqueur de pubs, vidage du cache… mais aussi passer sous Windows Phone). On entend moins parler des clients alternatifs. Ils sont pourtant très utiles pour les personnes cherchant un moyen simple d’accéder aux contenus YouTube sans être pisté, ou harcelé de pubs !
Comme le titre de la documentation le dit, nous allons ici parler de FreeTube, qui est mon client alternatif préféré sur PC. Allez, on passe directement dans le vif du sujet, voyons les fonctionnalités qui sont offertes par FreeTube.
Les fonctionnalités de FreeTube

  • Abonnements : Il vous sera possible de vous abonner à des chaînes sans avoir besoin d’un compte Google/YouTube.
  • Données locales : Tous les abonnements et l’historique sont stockés localement sur votre machine. Rien n’est envoyé à Google ou autre service de Cloud.
  • Pas de publicité : Profitez d’un visionnage épuré sans publicité. SponsorBlock est également activable, il permet d’ignorer les segments sponsorisés dans les vidéos YouTube.
  • Confidentialité : Vos habitudes de visionnage ne sont suivies ni par FreeTube ni par YouTube. FreeTube recueille les données en récupérant les informations dont il a besoin (à l’aide de méthodes locales ou en utilisant l’API Invidious).
  • Design similaire : Un design similaire à celui de YouTube facilite la transition vers FreeTube.
  • Source ouverte : FreeTube est un logiciel libre et open source sous licence AGPLv3.
  • Importer des abonnements : importez vos abonnements depuis YouTube pour voir votre flux instantanément.
  • Multiplateforme : Fonctionne sous Windows, Mac et plusieurs distributions Linux. Une version portable est également disponible !
  • Multilingue : FreeTube est traduit dans de nombreuses langues, dont le français.
    Autre fonction qui peut être utile, il est possible de télécharger directement une vidéo depuis l’interface, dans le format et la qualité souhaitée. Le PIP ( Picture-in-Picture ) est également disponible !
    Il faut aussi savoir que FreeTube est activement développé. Vous pouvez suivre cela sur leur dépôt sur GitHub À partir du lien suivant :

https://github.com/FreeTubeApp/FreeTube

Les nouvelles fonctionnalités/prise en charge sont régulières, tout comme les corrections de bugs. J’utilise actuellement la version 0.23.1 Beta en version instalable sous Windows 11 Pro, elle est tout à fait stable, je n’ai rencontré aucun souci pour le moment.
Pour configurer plus finement le comportement de FreeTube, rendez-vous dans les paramètres de l’application. Vous pourrez y modifier l’apparence et la langue de l’interface (français disponible), gérer vos abonnements et vos playlists, mais aussi régler des options avancées comme le bloqueur de sponsors ou le proxy réseau.
L’interface de FreeTube est en anglais par défaut.

Qu’est-ce que FreeTube ?

FreeTube est un client YouTube pour Windows (10 et versions ultérieures), Mac (macOS 10.15 et versions ultérieures) et Linux, conçu pour utiliser YouTube de manière plus privée.
Vous pouvez profiter de votre contenu et de vos créateurs préférés sans que vos habitudes ne soient suivies. Toutes vos données utilisateur sont stockées localement et ne sont jamais envoyées ni publiées sur Internet.
FreeTube récupère les données en récupérant les informations dont il a besoin avec des méthodes locales ou en utilisant éventuellement l’API Invidious).
Avec de nombreuses fonctionnalités similaires à YouTube, FreeTube est devenu l’une des meilleures méthodes pour regarder YouTube en privé sur un ordinateur de bureau.

Comment changer la langue de l’interface de FreeTube en français?

Comme je l’ai indiqué, l’interface de Freetube est disponible en anglais par défaut.
Si vous voulez le mettre par exemple dans notre belle langue en français, suivez les instructions suivantes:
1. Ouvrez le programme à partir de l’icône du bureau appelé:
FreeTube
2. Accédez au bouton appelé:
Settings
3. Une fenêtre s’ouvre comme titre (en anglais) :
Settings - FreeTube
4. Accédez au premier lien appelé:
General
Notre lecteur d’écran l’indique comme:
principale région liste de 15 éléments General lien
Appuyez sur Entrée pour l’ouvrir.
5. Rechercher ensuite le titre niveau 3 appelé:
General titre niveau 3
Faites plusieurs tabulations jusqu’à ce que vous trouviez l’élément appelé :
Locale Preference
Notre lecteur d’écran l’indique comme:
Locale Preference liste déroulante réduit
Faire barre d’espace pour le développer.
Notre lecteur d’écran l’indique comme:
développé
Par exemple, nous aurons les trois premières lignes annoncées comme:
Language preference liste déroulante System Default
Language preference liste déroulante English (US)
Language preference liste déroulante English (UK)
6. Pour obtenir la langue de l’interface de FreeTube en français, Tapez la lettre: f
Vous trouverez l’élément appelé:
Français (France
Notre lecteur d’écran l’indique comme:
Français (France) non sélectionné 20 sur 49
Faites entrée pour sélectionner cette langue.
Notre lecteur d’écran l’indique comme:
Préférence locale liste déroulante Français (France) réduit
Maintenant l’interface de FreeTube se trouve dans notre belle langue Le français!
7. Faites une fois de plus Tab Pour modifier le pays pour les tendances.
Par exemple, nous aurons le pays pour les tendances indiqué par notre lecteur d’écran comme:
Pays pour les tendances liste déroulante Azerbaïdjan réduit
Remarque, il est possible qu’un autre pays de tendances soit affiché pour vous.
Faire barre d’espace pour le développer.
Notre lecteur d’écran l’indique comme:
développé
8. Pour obtenir le pays de tendances de notre pays la France, Tapez la lettre: f
Vous trouverez l’élément appelé:
France
Notre lecteur d’écran l’indique comme:
France non sélectionné 33 sur 110
Faites entrée pour sélectionner ce pays.
Notre lecteur d’écran l’indique comme:
Pays pour les tendances liste déroulante France réduit
Maintenant le pays de tendances c’est notre pays la France!
Enjoy!
Maintenant, nous pouvons profiter de l’interface de FrreeTube en Français et le pays des tendances la France! 🇫🇷

Comment adapter un lien YouTube dans FreeTube ?

Si nous reprenons le lien utilisé pour Freetube dans l’exemple ci-dessus, s’il s’agissait de la page YouTube, le lien de l’exemple serait comme suit :
SALSA CLASICA VOL 15  SALSA ROMANTICA MIX | MEZCLADA EN VIVO POR DJ ADONI
https://www.youtube.com/watch?v=gGphx7a4V2Q

Dans le programme FreeTube, seule l’ID de la vidéo YouTube qui est à la fin du signe = de ladite vidéo devrait être insérée dans la zone d’édition.
Le début de l’URL YouTube n’est pas valable pour FreeTube dans ce cas :
https://www.youtube.com/watch?v=
Je devrais juste mettre l’ID se trouvant à la fin du signe = dans la zone de recherche :
gGphx7a4V2Q
Il s’agit de l’ID d’une vidéo YouTube , pas l’ID d’une chaîne YouTube à rechercher en FreeTube.
Voici le lien de la chaîne YouTube de DJ ADONI :
https://www.youtube.com/channel/UCg8e2UUwWg-d2jf3ryxZ9kg
Lorsque l’interface de FreeTube est ouverte, Si vous voulez consulter cette chaîne YouTube via FreeTube, mettez ceci dans la zone de recherche par exemple :
UCg8e2UUwWg-d2jf3ryxZ9kg
Pour les personnes qui ont le lecteur d’écran NVDA, ils doivent activer avec la touche Entrée le champ d’édition qui est annoncé en anglais comme suit : Search / Go to URL édition
Une fois fait, coller cet exemple dans ce champ d’édition et faire Entrée.
Une nouvelle page avec le résultat de la recherche s’ouvre.
Search Results
Trouvez le titre de niveau 3 appelé :
DJ ADONI
Ce titre de niveau 3 contient seulement le lien utilisé par FreeTube pour connaître le contenu de cette chaîne YouTube dans le programme FreeTube.
Appuyez sur Entrée pour le consulter.
Lorsque cette page sera ouverte via FreeTube, vous trouverez divers liens comme titre de niveau 3 (avec le nom de ladite vidéo YouTube associé à la chaîne YouTube).
Dans le programme FreeTube, seule l’ID de la chaîne YouTube qui est à la fin de la barre oblique de ladite chaîne devrait être insérée dans la zone d’édition.
Le début de l’URL d’une chaîne YouTube n’est pas valable pour FreeTube dans ce cas :
https://www.youtube.com/channel/
Je devrais juste mettre l’ID se trouvant à la fin de la barre oblique dans la zone de recherche :
UCg8e2UUwWg-d2jf3ryxZ9kg

Et si vous souhaitez écouter le contenu de chaque vidéo YouTube, suivez les étapes indiquées ci-dessus.

Lorsque le programme FreeTube est ouvert vous pouvez commencer à explorer les contenus en tapant des mots-clés, exactement comme sur le site officiel dans la zone de recherche.
