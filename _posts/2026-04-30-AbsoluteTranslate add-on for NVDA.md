Coucou mes amis du blog de BlindHelp!    
Aujourd'hui, je vous présente une nouvelle extension pour notre lecteur d'écran NVDA appelée: AbsoluteTranslate.    

Pourquoi j'ai fait ce choix ?    

Une vilaine histoire. Je voulais faire fonctionner l'extension pour traduire appelée TranslateAdvanced (Traducteur Avancé) extension pour NVDA, mais ça me dit que c'est une extension incompatible, mais je l'ai installée pour la tester avec la nouvelle version de NVDA 2026.1.

Par contre J'ai remarqué que cette extension n'apparaît pas dans les gestes de commande, et je viens de la désinstaller.

En revanche, j'ai trouvé Absolute translate qui est peut-être son successeur. En voici la description ci-dessous. Merci.

Vous trouverez cette extension à partir de l'Add-on Store.

L'Add-on Store est accessible depuis le sous-menu Outils du menu NVDA. Pour accéder à l'Add-on Store depuis n'importe où, attribuez un geste personnalisé à l'aide de la boîte de dialogue Gestes de commandes.

Pour rechercher cette  extension, utilisez la zone de texte "Rechercher". Vous pouvez y accéder en appuyant sur maj+tab dans la liste des extensions. Tapez un ou deux mots-clés pour le type d'extension que vous recherchez, dans notre cas c'est AbsoluteTranslate puis tabulation pour aller à la liste des extensions. L'extensions sera répertoriée si le texte de recherche peut être trouvé dans l'ID, le nom, l'éditeur l'auteur ou la description de l'extension.

Sélectionnez l'extension AbsoluteTranslate dans l'onglet "Extensions disponibles" ensuite ces actions sont accessibles via un menu qu'on ouvre en appuyant sur la touche applications, entrée, un clic droit ou un double-clic sur l'extension AbsoluteTranslate. Ce menu est également accessible par un bouton Actions dans les détails de l'extension sélectionnée.

Voici les informations en français trouvés dans l'Add-on Store à propos de cette extension:
