Aujourd'hui, je vous présente une nouvelle extension pour notre lecteur d'écran NVDA appelée: translate.    

Avertissement: 💀    
Le blog de BlindHelp n'est pas responsable des dommages causés par une mauvaise utilisation de l'extension translate téléchargé ni des informations ce trouvant sur la documentation dédié et l'utilisation de l'extension translate téléchargé est à vos risques et périls. ☠

Vous trouverez cette extension à partir de l'Add-on Store.

L'Add-on Store est accessible depuis le sous-menu Outils du menu NVDA. Pour accéder à l'Add-on Store depuis n'importe où, attribuez un geste personnalisé à l'aide de la boîte de dialogue Gestes de commandes.

Pour rechercher cette  extension, utilisez la zone de texte "Rechercher". Vous pouvez y accéder en appuyant sur maj+tab dans la liste des extensions. Tapez un ou deux mots-clés pour le type d'extension que vous recherchez, dans notre cas c'est translate puis tabulation pour aller à la liste des extensions. L'extensions sera répertoriée si le texte de recherche peut être trouvé dans l'ID, le nom, l'éditeur l'auteur ou la description de l'extension.

Sélectionnez l'extension translate dans l'onglet "Extensions disponibles" ensuite ces actions sont accessibles via un menu qu'on ouvre en appuyant sur la touche applications, entrée, un clic droit ou un double-clic sur l'extension translate. Ce menu est également accessible par un bouton Actions dans les détails de l'extension sélectionnée.

Voici les informations en français trouvés dans l'Add-on Store à propos de cette extension:

# Description :

Un module complémentaire de traduction en ligne ultra-rapide et léger pour NVDA. Créée à l'origine par Yannick Plassiard, cette version modifiée a été optimisée pour des traductions rapides en temps réel à l'aide de moteurs cloud à grande vitesse. Des remerciements particuliers et tout le crédit vont au développeur d'origine pour son travail exceptionnel. Nécessite une connexion Internet.
