Une fois notre lecteur d'écran NVDA lancé et vous souhaitez utiliser son extension appelée FreeRadio, procédez comme suit:    

👉 À savoir : Cette interface de ladite extension est traduit en français à partir de la version 2026.19.7, cependant, mes explications ci-dessous concernent la version précédente, c'est-à-dire la version 2026.19.6 qui est en anglais bien sûr.    

Pour ouvrir l'interface de cette extension on atteint plus rapidement le menu en frappant Windows + Ctrl + R. Là, on se trouve par défaut dans le premier onglet appelé: `All Stations`    

Les autres onglets portent les noms :    
`Favourites`    
`Recording`    
`Timer`    
`Liked Songs`     

Pour naviguer entre les onglets, utilisez les touches Ctrl+Tab    
Si nous appuyons sur la touche de tabulation, nous trouverons plusieurs éléments en fonction de l'onglet focalisé.    

Ce  tutorial rapide pour l'extension FreeRadio ne traitera que du premier onglet appelé: `All Stations`    

Si nous appuyons sur la touche de tabulation voici les valeurs qui viennent par défaut dans ledit onglet :    

`Country: liste déroulante All réduit`    
Si vous avez laissé cette option par défaut, toutes les stations de tous les pays seront affichées.    
Vous pouvez changer ce mode en recherchant dans la liste le pays souhaité, par exemple la France    
Pour ce faire, tapez plusieurs fois la lettre f pour trouver le nom du pays.    
`Country: France`    
Faire Tabulation.    
`Search: édition ligne 1 vide`    
Ici dans ce champ d'édition, vous pouvez écrire le type de recherche voulu, ensuite faire Entrée.    
Par exemple: jazz, salsa, Etc.    
Cette recherche prend en compte le contenu que vous souhaitez rechercher si vous avez choisi un pays spécifique ou si vous l'avez laissé par défaut en `All`    
Faire Tabulation.    
`Stations: liste`    
`adofm - France - dance`    
Vous pouvez utiliser la flèche haut ou bas pour circuler entre les radios de ce pays ou toutes les radios de tous les pays si vous n'avez rien modifié.    
Vous pouvez appuyer sur la touche Entrée ou sur la barre d'espace pour écouter cette radio sélectionnée ou sur la barre d'espace pour mettre la radio sélectionnée en pause.    
Vous pouvez appuyer sur la flèche gauche ou droite pour activer une des radios répertoriées dans la liste pour ce pays et la mettre  en pause avec la barre d'espace et appuyez à nouveau sur la barre d'espace pour la rejouer.    
Faire Tabulation.    
`Output device: liste déroulante System default réduit`    
Ici, vous pouvez modifier le périphérique de sortie.    
Utilisez la flèche haut ou bas pour choisir le périphérique de sortie.    
Faire Tabulation.    
`Volume: édition sélectionné 100`    
Flèche haut ou bas pour modifier la valeur du volume par palier d'un pour cent.    
Vous pouvez également écrire la valeur pour modifier le volume, par exemple : 30 puis faire tabulation pour que cela prenne effet.    
Si vous n'avez rien modifié dans la valeur du volume faire Tabulation.    
`Effects: liste`    
`Chorus non sélectionné`    
Utilisez la flèche haut ou bas pour choisir l'effet souhaité dans la liste des effets lors de l'écoute de la radio.    
Faire Tabulation.    
`Play/Pause bouton Alt+ p`    
Appuyez sur ce bouton pour écouter ou mettre en pause la radio focalisé sur la liste des radios.    
Faire Tabulation.    
`Add Custom Station... bouton Alt+ u`    
Ce bouton est utilisé pour ajouter une station personnalisée. (je n'ai pas essayé).    
Faire Tabulation.    
`Close bouton Alt+ c`    
Appuyez sur ledit bouton pour fermer la boîte de dialogue appelée: `FreeRadio - Station Browser`    
Veuillez noter que dans ce dialogue, échap fonctionne de la même manière que le bouton Close.    

Remarque : Lorsque vous avez sélectionné une radio dans la liste et que vous faites Tabulation, vous obtiendrez deux boutons supplémentaires appelés :    
`Add to Favorites bouton Alt+ v`    
Ce bouton permet d'ajouter la radio à une liste de favoris.    
`Station Details bouton Alt+ l`    
Ce bouton est très utile puisque si vous appuyez dessus vous obtiendrez les détails de la radio sélectionnée, comme le nom de la station, le pays, la langue, le genre, le Format par exemple: MP3, 320 kbps, l'URL de la Website, le Stream URL ; c'est-à-dire le flux d'écoute de la radio, le nombre de votes que la radio a.    
Faire Tabulation puis utilisez les touches fléchées pour parcourir les détails de la radio sélectionnée.    
Ces champs d'édition sont en lecture seule et ils ne sont pas modifiables.    
La cerise sur le gâteau, en faisant Tabulation, vous trouverez le bouton appelé :    
`Copy all to clipboard bouton Alt+ c`    
Une fois ce bouton activé, toutes les informations de la station de radio seront copiées dans le presse-papiers.    
De cette façon, vous pouvez le partager avec vos amis.    
Faire Tabulation.    
Appuyez sur le bouton Ok pour fermer la boîte de dialogue contenant les détails de la station de radio.    
Veuillez noter que dans ce dialogue, échap fonctionne de la même manière que le bouton OK.    

Ensuite, vous vous retrouvez dans le premier onglet actif appelé: `All Stations`.    

👉 À savoir : Si vous n'avez pas fermé la session actuelle de NVDA, la recherche que vous avez effectuée dans le premier onglet sera conservée, mais si vous fermez la session actuelle de NVDA celle-ci sera perdu.    
Seul le changement de volume sera conservé.    

Mais je pense que ceci n'est pas grave puisque l'extension fonctionne avec l'API en directe lors du chargement de ladite extension au démarrage de NVDA.    

Vous trouverez l'aide (en anglais) de cette extension à partir de l'Add-on Store sous le nom de l'extension.    
`freeRadio; État: Activée; Version installée: 2026.19.5; Canal: Stables; Auteur: Çağrı Doğan <cagrid@hotmail.com>; Date de publication: 18/05/2026; Date d'installation: 19/05/2026; Version minimale de NVDA: 2024.1; Dernière version NVDA testée: 2026.1`    
Tabulez et appuyez sur le bouton appelé :    
`Actions bouton Alt+ a`    
`Contexte menu`    
Puis recherchez et appuyez sur le bouton appelé :    
`Aide e`    

L'aide en anglais pour ladite extension s'ouvrira dans votre navigateur par défaut.    

👉 À savoir : Nous retrouverons les gestes de commandes  utilisés pour l'extension dans la boîte de dialogue Gestes de commandes, pour ce faire,  aller à:    

`NVDA menu`    
`Préférences sous-Menu p`    
`Paramètres... p`    
Trouvez l'élément appelé :    
`Gestes de commandes... g`    
Faire Entrée puis Une boîte de dialogue s'ouvre.    
`Gestes de commandes dialogue`    
Appuyez sur la lettre f pour avoir le nom de l'extension appelée : FreeRadio    
NVDA vous l'annoncera comme suit :    
`FreeRadio réduit 16 sur 36 niveau 0`    
Faire une flèche gauche pour le développé    
NVDA vous l'annoncera comme suit :    
`développé`    
Et puis vous retrouverez les 13 éléments contenant les gestes  de commandes répertorié comme suit:    
NVDA vous l'annoncera comme suit :    
`13 éléments`    
Utilisez les flèches haut et bas pour vous déplacer entre les éléments.    
* `Add currently playing station to favourites`    
windows+contrôle+v (clavier, toutes les dispositions)    
* `Announce currently playing station. Press twice for full details, three times to copy track info, four times to force music recognition.`    
windows+contrôle+i (clavier, toutes les dispositions)    
* `Decrease FreeRadio volume by 10`    
windows+contrôle+flèche bas (clavier, toutes les dispositions)    
* `Increase FreeRadio volume by 10`    
windows+contrôle+flèche haut (clavier, toutes les dispositions)    
* `Mirror audio to an additional output device`    
windows+contrôle+m (clavier, toutes les dispositions)    
* `Open FreeRadio recordings folder`    
contrôle+windows+w (clavier, toutes les dispositions)    
* `Open FreeRadio station browser`    
contrôle+windows+r (clavier, toutes les dispositions)    
* `Pause or resume FreeRadio playback`    
windows+contrôle+p (clavier, toutes les dispositions)    
* `Play next station`    
windows+contrôle+flèche droite (clavier, toutes les dispositions)    
* `Play previous station`    
contrôle+windows+flèche gauche (clavier, toutes les dispositions)    
* `Start or stop instant recording`    
windows+contrôle+e (clavier, toutes les dispositions)    
* `Stop FreeRadio playback`    
windows+contrôle+s (clavier, toutes les dispositions)    
* `Toggle mute notifications (station changes, playback, recording, volume level)`    
Veuillez assigner un geste de commande personnalisé en utilisant le dialogue Gestes de Commandes.    

👉 À savoir : Nous retrouverons les options suivantes de ladite extension, pour cela faites :    
Appuyez sur NVDA+n pour ouvrir le menu NVDA, puis sur o pour Outils, puis sur f pour FreeRadio. Lorsque FreeRadio s'ouvre, il affiche les éléments suivants:    
`Station Browser...	Ctrl+Win+R b`    
En appuyant sur cet élément ouvrira la première boîte de dialogue que nous avons expliquée au début de cet article.    
`FreeRadio Settings... s`    
En appuyant sur cet élément ouvrira les paramètres de ladite extension, mais la vérité est que je n'y ai pas du tout touché.    
`Check for Updates... u`    
Vous pouvez également vérifier manuellement les mises à jour de ladite extension en appuyant sur cet élément.    

Dans tous les cas, je vous laisse découvrir par vous-même la possibilité d'écouter vos radios et de voir les autres choses que propose l'extension FreeRadio.
