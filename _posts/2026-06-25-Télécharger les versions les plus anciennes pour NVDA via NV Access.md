Ci-dessous je vous donne un récapitulatif  des liens officiels pour télécharger les versions les plus anciennes pour NVDA via NV Access    

Remarque : NVDA n'est disponible que pour les PC fonctionnant sous Microsoft Windows 10 et versions ultérieures.    

Si vous avez besoin d'une version de NVDA qui peut encore fonctionner sous Windows 8.1 ou les versions 32 bits ou ARM de Windows 10, veuillez télécharger l'ancienne version NVDA 2025.3.3 pour Windows 8.1 à la place.    

Vous pouvez consulter le lien ci-dessous qui vous mènera à la page web en anglais pour cette version :    

<https://www.nvaccess.org/download/?nvdaVersion=2025.3.3>    

[Téléchargement direct de NVDA 2025.3.3 à partir du serveur de NV Access (lien officiel)](https://www.nvaccess.org/download/nvda/releases/2025.3.3/nvda\_2025.3.3.exe)

Si vous avez besoin d'une version de NVDA qui peut encore fonctionner sous Windows 7, veuillez télécharger l'ancienne version NVDA 2023.3 pour Windows 7 à la place.    

Vous pouvez consulter le lien ci-dessous qui vous mènera à la page web en anglais pour cette version :    

<https://www.nvaccess.org/download/?nvdaVersion=2023.3>    

[Téléchargement direct de NVDA 2023.3 à partir du serveur de NV Access (lien officiel)](https://www.nvaccess.org/download/nvda/releases/2023.3/nvda\_2023.3.exe)

Si vous avez besoin d'une version de NVDA qui peut encore fonctionner sous Windows XP ou Vista, téléchargez plutôt l'ancien NVDA 2017.3 pour Windows XP.    

Vous pouvez consulter le lien ci-dessous qui vous mènera à la page web en anglais pour cette version :    

<https://www.nvaccess.org/download/?nvdaVersion=2017.3>    

[Téléchargement direct de NVDA 2017.3 à partir du serveur de NV Access (lien officiel)](https://www.nvaccess.org/download/nvda/releases/2017.3/nvda\_2017.3.exe)

NV Access ne recommande pas et ne soutient pas l'exécution de ces anciennes versions sur des systèmes d'exploitation plus récents.

Sur ce, je vous souhaite une très bonne utilisation DE votre lecteur d'écran NVDA gratuit et Open-source pour Microsoft Windows! :)
