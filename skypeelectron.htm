Utilisation des scripts JAWS pour Skype pour Windows Desktop 8 (Electron)
Dernière révision en juillet 2018
La traduction à été fait par Rémy Ruiz (BlindHelp) en juillet 2018.

Ce guide fournit des conseils pour utiliser Skype pour Windows Desktop version 8 avec JAWS pour Windows et ces scripts. Ces scripts et cette documentation s'appliquent à Skype pour Windows Desktop 8 (une app Electron). Dans ce document, cette version de Skype sera simplement appelée Skype 8. Lorsque les scripts sont installés et que Skype 8 est ouvert et mis en focus, ce document peut être ouvert via un double appui sur TOUCHE JAWS+F1 (ou Insert+F1).

Configuration système requise pour les utilisateurs de JAWS

Ces scripts nécessitent JAWS 17.0 ou supérieur. (Les versions de JAWS antérieures à 17.0 ne supporte pas les scripts spécifiques aux sites Web, et Skype 8 s'exécute dans la fenêtre Google Chrome.)

Instructions d'installation du script

Pour installer ces scripts sur un nouveau système:

  1. Charger JAWS si cela n'a pas déjà été fait. Cela nécessitera des privilèges administrateur sur l'ordinateur.
  2. Exécutez JAWS en tant qu'utilisateur pour lequel les scripts doivent être installés. Ceci et les étapes suivantes doivent être effectuées pour chaque utilisateur de l'ordinateur qui utilisera JAWS avec ces scripts.
  3. Exécutez le programme d'installation pour ces scripts et suivez les instructions à l'écran. Veillez à installer les scripts dans la Version JAWS si une liste de versions JAWS est présentée.
  4. Pour vérifier la réussite de l'installation, tapez Insert+Q pendant que Skype 8 est mis en focus. Une partie de la réponse verbale de JAWS devrait être un numéro de révision. Si vous n'entendez pas de numéro de révision, les scripts ne sont pas correctement chargés.

Utiliser Skype 8 avec JAWS et les scripts

Cette section et ses sous-sections fournissent des astuces pour utiliser Skype 8 avec JAWS et ces scripts.

Raccourcis clavier natifs de Skype 8

Skype 8 fournit plusieurs raccourcis clavier. Celles-ci fonctionnent avec ou sans JAWS ou ces scripts. Microsoft maintient une liste des raccourcis natifs sur la page en anglais "What are hotkeys and how do I use them in Skype?" cette page en anglais , qui répertorie également les raccourcis clavier pour d’autres versions de Skype sur Windows et d’autres systèmes d’exploitation. Veillez à sélectionner le bon onglet ("Hotkeys in Skype for Windows Desktop") ("Raccourcis clavier dans Skype pour Windows Desktop") pour cette version de Skype. Les onglets se trouvent sous le tableau de raccourcis clavier situé au bas de la page (en anglais).

Commandes et fonctionnalités du script JAWS

La plupart des commandes de script Skype 8 sont des commandes par frappes successives (commandes séquentielles), plutôt que des combinaisons multi-touches. Cela permet d'éviter les collisions entre les commandes par frappes successives (commandes séquentielles), du script et les séquences de touches d'application natives. Toutes ces séquences de touches commencent par une touche de préfixe commune, crochet ouvert ([) par défaut. Les utilisateurs peuvent modifier la touche de préfixe si nécessaire en tapant crochet ouvert [ suivi par TOUCHE JAWS+C ou Insert+C. Le changement survivra à travers le script et les mises à jour de JAWS. Dans ce document, la touche de préfixe s'appellera Commande; donc, par défaut, Commande e; signifie qu'il faut frappez crochet ouvert [ suivi de e.

Les scripts fournissent les commandes suivantes:

  • Commande e: déplace le focus sur le champ de saisie de discussion (chat en anglais) pour la conversation affichée, s'il en existe une.
  • Commande h: déplace le focus sur la liste de l'historique des messages de conversation de chat affichés.
  • Commande t: annonce qui est entraint d'écrire dans la conversation affichée.
  • Commande w: ouvre la page d'accueil de ces scripts (en anglais) dans le navigateur Web par défaut.
  • Insert+F1: saisie deux fois rapidement, ouvre ce document dans le navigateur par défaut.
Notez que le script Alt+1 pour focaliser la liste des conversations a été supprimé le 17 juillet 2018, en l'honneur de Microsoft en implémentant correctement cette fonctionnalité dans l'app par la version 8.25. Alt+1 fonctionne maintenant avec et sans ces scripts.

Les scripts fournissent également les fonctionnalités suivantes:

  • Le curseur virtuel JAWS est désactivé à chaque fois que Skype 8 prend le focus, pour simplifier la navigation. Utiliser Insert+Z, dans Skype si vous souhaitez activer ou rédésactiver le curseur virtuel manuellement.
  • TOUCHE JAWS+Q, tout au long avec l'annonce du nom de la configuration active, annoncera le numéro de révision de ces scripts.
  • La commande JAWS SayWindowTitle, Insert+T, comprend le nombre de messages non lus Lorsque sont affichés, et aussi le nom de la conversation active si elle existe, ou sinon "Home". Lorsqu'une conversation est affichée et que de telles informations sont disponibles, les informations de présence du contact de la conversation et / ou le message d'humeur sont également inclus.

Conseils utiles

Quand un appel arrive dans Skype 8, signalé par le son de l'appel entrant, ayant le focus sur Skype et ensuite Tab pour voir les informations sur le correspondant (Autrement dit l'appelant), puis les boutons pour gérer l'appel.

Utiliser Alt+1 pour déplacer le focus sur la liste de conversation. C'était Alt+2 dans le Skype classic pour Windows Desktop.

Appuyez sur Entrée sur une conversation dans la liste de conversation pour afficher la conversation et placer le focus dans le champ de saisie correspondant. Dans le cas où le focus ne se pose pas dans le champ de saisie, utilisez la commande séquentielle scripté, Commande e, pour y déplacer le focus.

Utiliser Alt+2 pour ouvrir la liste de contacts. C'était Alt+1 dans le client Skype classic pour Windows Desktop. Puisque la liste de contacts est recherchée par la boîte de dialogue "Rechercher dans Skype" (rapidement accessible avec Ctrl+F), cela peut rarement être nécessaire d'ouvrir la liste de contacts directement.

Un moyen rapide de commencer une nouvelle conversation:

  • Appuyez sur Ctrl+F pour y déplacer le focus sur la boîte de dialogue "Rechercher dans Skype".
  • Tapez le nom ou le nom partiel de votre contact prévu, et ensuite Tab deux fois pour atteindre les résultats de la recherche.
  • Utiliser les flèches Haut et Bas pour trouver votre contact prévu. Si, au début, vous ne trouvez pas le contact, vous parcourez à nouveau les résultats car la liste peut être mise à jour au fur et à mesure que d'autres correspondances sont trouvées.
  • Appuyez sur Entrée pour ouvrir une conversation et déplacer le focus dans le champ de saisie. Si le focus ne se pose pas dans le champ de saisie, utilisez la séquence scriptée, Commande e, pour y déplacer le focus.
  • Pour appeler le contact immédiatement une fois la conversation affichée, tapez Ctrl+Maj+P pour un appel vocal ou Ctrl+Maj+K pour un appel vidéo.

Avertissement: Les raccourcis clavier Ctrl+Maj+P et Ctrl+Maj+K pour faire des appels vocaux et vidéo appellera le contact affiché, pas le contact dont l'entrée est focalisée sur la liste de conversation s'il s'agit d'un contact différent. Pour être sûr que vous êtes en train d'appeler le contact correct tout en naviguant dans la liste de conversation, appuyez sur Espace ou Entrée avant de faire un appel.

Problèmes connus

Les problèmes présentés dans cette section peuvent être résolus dans les futurs révisions de scripts et / ou applications. mais ne sont pas traités à ce jour.

Il n'y a pas le système habituel dans ces scripts pour naviguer parmi les messages de discussion (chat en anglais). Historiquement, cela a été fait avec Alt avec des chiffres pour lire les messages récents, et Alt avec les flèches, Origine, Fin, et +5 du pavé numérique pour lire séquentiellement tout en gardant une trace de la position actuelle. La liste des messages de discussion (chat en anglais) de Skype 8 ne semble pas offrir ce type de navigation.

Au moins dans les anciennes versions de Skype 8 et / ou des révisions de script, les commandes tels que Commande h, pour sauter à des contrôles spécifiques peut parfois déplacer le focus sur le mauvais contrôle. Si cela se produit, essayez à nouveau la commande après avoir tapé TOUCHE JAWS+T, deux fois successivement, ce qui efface les notions du script d'où les contrôles sont situés ou seront-ils scannés pour cette information. Ce problème n'a pas été vu récemment en mai 2018 et Skype 8.25.

Ouvrir ce document de l'intérieur de Skype 8 en tapant Insert+F1 deux fois successivement peut laisser le curseur virtuel JAWS inactif. Utilisez Insert+Z pour réactiver le curseur virtuel si cela se produit.
